# Чехов — заказы

Офлайн PWA для официанта **Гастропаб Чехов** (Ярославль).

## Возможности

- Столы и активные заказы
- Меню из методички: описание, состав, аллергены + цены с [menu.chehovgastropub.ru](https://menu.chehovgastropub.ru/ru/restaurant/)
- Отметки «подано» / «оплачено»
- Статистика смены
- Работает офлайн (PWA + IndexedDB)

## Запуск

```bash
npm install
npm run dev
```

Откройте на телефоне/планшете в той же Wi‑Fi сети и добавьте на главный экран.

## Обновление меню

Если изменилась методичка или цены на сайте:

```bash
python scripts/build-menu.py
```

## Сборка

```bash
npm run build
npm run preview
```
